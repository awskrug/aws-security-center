    로컬 파트너

    보안 관련 한국 파트너 업체에 대한 정보입니다. 아래 목록은 AWS Korea 파트너팀에서 관리하고 있는 목록으로서, 한국의 고객들이 영업문의나 기술지원을 받을 수 있는 업체를 기준으로 하고 있습니다.

    본 목록의 내용은 각 업체의 솔루션이나 제품에 대한 보증을 AWS가 하는 것이 아님을 분명히 밝히며, 또한 나열된 순서는 제품의 품질이나 AWS가 추천하는 순서와 전혀 상관이 없습니다. 고객분들은 반드시, 제품/솔루션에 대한 검증을 수행하실 것을 권장드립니다.

    AWS 보안관제 및 컨설팅

    AWS 클라우드 관제

    • 안랩 : 클라우드 컴퓨팅 보안 관제
    • SK 쉴더스 : 클라우드 원격 보안 관제

    AWS WAF 관제

    • SK 쉴더스 : AWS WAF 관제 - AWS WAF + Managed 룰셋 기반 관제

    보안 컨설팅

    • SK 쉴더스 : 클라우드 보안 컨설팅 - 보안 컨설팅, 24x365 운영, SI 구축 등

    사용자 보안

    AWS 콘솔 Single Sign On

    • 이니텍 : INISAFE Nexess - SSO
    • NETS - IAM (AWS IAM 연계)
    • CA : AWS API Proxy _- CA PAM(Privileged Access Manager)와 AWS 콘솔 간 SSO 연계

    서버 접근제어(OS 접근제어)

    • 넷엔드 : HIWARE - DB접근제어 + OS접근제어
    • PNP Secure : DBSafer OS
    • 시큐브 : iGRIFFIN - OS접근제어

    CASB(Cloud Service Access Broker)

    • 시만텍 : CASB - 예전 Bluecoat CASB 제품

    네트워크 보안

    방화벽, 침입탐지/침입방지(IDS/IPS), UTM

    • 안랩 : vTrusGuard for AWS - G/W타입 UTM, anti-malware 포함, Auto-scaling 지원
    • 체크포인트 : CloudGuard IaaS - G/W 타입 UTM
    • 포티넷 : FortiGate-VM - G/W 타입 UTM, Auto-scaling 및 Transit VPC 지원
    • 팔로알토 : VM Series Next-Gen F/W - G/W 타입 UTM, Auto-scaling 및 Transit VPC 지원
    • 소포스 : UTM 9 Autoscaling - G/W 타입 UTM, WAF 기능 포함, Auto-scaling 및 Out-Bound G/W 지원
    • 트렌드 마이크로 : Deep Security - Hosted IDS/IPS(Agent 탑재형), anti-malware 포함, Auto-scaling 지원
    • F5 : Big-IP VE - G/W 타입 IPS

    웹 방화벽

    • F5 : WAF
    • 모니터랩 : WIWAF-SE - SW 설치형, AIONCLOUD - SaaS형, WAF + 악성코드 탐지
    • 펜타 시큐리티 : WAPPLES SA (EC2 설치형), CloudBric WAF+ (SaaS형)
    • 소포스 : UTM 9 Autoscaling - G/W 타입 UTM, WAF 기능 포함

    망분리

    • HUNESION : i-oneNet - 망연계(자료전송)
    • Softwith : CrossNet - 망연계(자료전송 / 스트림 연계)

    보안 게이트웨이

    • Zscaler : ZIA(Zscaler Internet Access) / ZPA(Zscaler Private Access)

    DB 보안

    DB 암호화

    • 펜타 시큐리티 : D'AMO - EC2 설치형 DB만 지원, RDS의 경우 D'Amo BA(API방식)방식을 활용
    • 이글로벌 : CubeOne - EC2 설치형 DB만 지원, RDS의 경우 CubeOne API방식을 활용해야 됨
    • Spiceware - RDS와 EC2 설치형 DB 모두 지원, Java EE 와 Spring 환경에 Filter 방식으로 Agent를 이용하여 Application 변경 없이 지원 함
    • K-Sign : KSign Secure DB - EC2 설치형 DB만 지원(Agent방식, 파일단위 암호화, 애플리케이션 수정 필요없음), RDS의 경우 SecureDB API방식을 활용해야 됨(애플리케이션 수정 필요)

    DB 접근제어

    • PNP Secure : DBSafer DB
    • 신시웨이 : PETRA
    • 웨어밸리 : Chakra Max
    • 소만사 : DB-i
    • QueryPie : Simplified Data Access Control - RDS, DDB 접근제어 지원 + Automated Auditing & Reporting - SQL 로그분석, 접속기록 감사

    암호화 / 마스킹 / 익명화 / 개인정보

    암호키 관리

    • 펜타 시큐리티 : D'AMO KMS

    암호화 SDK

    • 펜타 시큐리티 : D'AMO Cloud - 파일/폴더/DB암호화/어플리케이션Data에 사용 가능

    암호화 게이트웨이

    • THALES : Vormetric Cloud Encryption G/W - 암호화 게이트웨이 S3 지원

    파일, 볼륨 암호화

    • THALES : Vormatric Transparent Encryption - S3, EBS, EFS, EC2 설치형 DB 볼륨, Docker 환경 지원

    DRM

    • Fasoo : Enterprise DRM for Repository - 서버 DRM
    • INKA Entworks : PallyCon multi-DRM Cloud - SaaS 형태

    마스킹 / 익명화

    • THALES : Vormetric Data Security Platform - Virtual Appliance, 토큰화, 마스킹, 변환, 파일 단위 암호화
    • 파수닷컴 : AnalyticDID
    • QueryPie : Enhanced Data Privacy - 마스킹, 접속기록, 경보, 접근사유 기록 관리

    개인정보 탐지

    • 소만사 : Server-i
    • QueryPie : Enhanced Data Privacy _- 개인정보 접근 승인 워크플로/이력관리 _

    서버 보안

    안티 바이러스 - 멀웨어

    • 안랩 : V3 Net for Windows / Linux
    • 트렌드 마이크로 : Deep Security - Hosted IDS/IPS(Agent 탑재형), anti-malware 포함, Windows/Linux 용
    • 멕아피 : Cloud Workload Security - 위변조 방지 기능 포함, Windows/Linux 용
    • 시만텍 : Protection Engine for Cloud Services - Windows/Linux 용

    안티 웹쉘

    • umv : Web Server Safeguard
    • SK 쉴더스 : Anti-Webshell - Non-Agent 기반
    • SSR : MetiEye -쉘 모니터
    • 에스큐브아이 : ShellCop

    로그통합 / SIEM

    로그 통합

    • 스플렁크 : Splunk Enterprise

    SIEM

    • Micro Focus : ArcSight Enterprise Security Manager / Logger
    • IBM : QRadar ALB Access Log / CloudTrail DSM / VPC Flow Logs

    기타 솔루션 영역

    이메일 보안

    • Barracuda : Email Security Gateway
    • 소만사 : Mail-i Cloud - SaaS 타입

    모바일 애플리케이션 보호

    • INKA Entworks : AppSealing Mobile Game Protection - SaaS 형태, 모바일 게임 앱에 대한 불법 해킹 등에 대처

    인증 지원

    • SAK(한국 정보 보호 인식) - K-ISMS-P, GDPR, CSP 안전성평가 등 지원
