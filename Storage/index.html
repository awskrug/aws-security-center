    EFS

    Amazon EFS(Amazon Elastic File System)는 AWS 클라우드 서비스와 온프레미스 리소스에서 사용할 수 있는, 간단하고 확장 가능하며 탄력적인 완전관리형 NFS 파일 시스템을 제공합니다. EFS의 보안관련 기능들은 아래와 같습니다.

    Amazon FSx for Windows File Server
